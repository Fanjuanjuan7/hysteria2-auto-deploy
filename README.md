# Hysteria2 一键部署脚本

适用于跨境电商使用的 Hysteria2 协议节点部署，支持端口跳跃、混淆、自签名证书、Nginx反向代理、分享链接生成等功能。

系统名称	版本建议	支持情况
Ubuntu	20.04、22.04、24.04	✅ 完全支持
Debian	10、11、12	✅ 完全支持
Linux Mint	基于 Ubuntu 的版本	✅ 支持

在服务器上运行以下命令：
